# Fixing a linear issue

## Process

1. Read issue
2. Fix issue (ask questions if applicable)
3. Ask user for manual review by prompting "I'm done with implementing the issue. Feel free to review. When you give me green light I will commit and move the issue to "Done"
4. Run the /commit command @.claude/commands/commit.md
5. Move the issue to "Done"

If "pr" is appended also: 6. Create a PR 7. Link the issue to the PR
