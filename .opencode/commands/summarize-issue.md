# Summarize an unresolved issue for handoff

## Process

1. State the observed user-facing problem in one or two sentences.
2. List the exact files and components involved.
3. Describe what currently works and what does not work.
4. Include relevant implementation context and recent attempted changes.
5. Include known verification results, errors, or blockers.
6. Do not propose a fix unless the user explicitly asks for one.

## Output format

Use concise markdown with these sections:

- **Problem**
- **Relevant Files**
- **Current Behavior**
- **Known Working Comparison**
- **Implementation Context**
- **Attempts Already Made**
- **Verification / Constraints**

Keep it paste-ready for another agent. Avoid speculation beyond clearly labeled uncertainty.
